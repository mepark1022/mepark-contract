# ME.PARK 근로계약서 관리 시스템

**(주)미스터팍** 근로계약서 관리 웹앱

## 기능

- 🔐 **관리자 인증** — Supabase Auth 기반 로그인/회원가입
- ✉️ **관리자 초대** — 이메일 초대 + 역할(슈퍼관리자/일반관리자/뷰어)
- 📊 **대시보드** — 재직현황, 수습알림, 사업장별 인원, 근무유형 분포
- 👥 **직원대장** — CRUD + 필터/검색 (사업장/근무유형/상태/신고유형)
- 📝 **계약서** — 평일제(11조)/주말제(10조) + 임금테이블 자동산출 + 수습기간 + A4 인쇄
- 📋 **사직서** — 사유 8종 프리셋 + 30일 미만 경고 + A4 인쇄
- 📄 **재직증명서** — 직원선택 → 자동채움 → A4 인쇄
- ⚙️ **설정** — 계약서 조항 편집 + 거래처/근무형태 코드표

## 기술 스택

- React 18 + Vite
- 인라인 스타일 (Tailwind 미사용)
- Noto Sans KR 폰트
- Vercel 배포

## 개발

```bash
npm install
npm run dev
```

## 빌드 & 배포

```bash
npm run build   # dist/ 폴더 생성
```

Vercel 자동 배포: GitHub push → Vercel 자동 빌드
